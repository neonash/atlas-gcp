    Instruction
    • This chart can be used to visualize data from a pivot table.
    • The most useful feature is the possibility of quickly changing the portion of data displayed.
    • For example, a store owner might list monthly sales totals for a large number of merchandise items in a table. If the owner wanted to know which items sold better in a particular financial quarter, it would be very time-consuming to look through multiple pages of figures to find the information. A pivot table would allow the owner to quickly reorganize the data and create a summary for each item for the quarter in question.
    • A review/conversation can be tagged under multiple dimensions. Hence, sum of the values might not line up with the actual totals.
    • The renderer type can be selected from a choice of values (see dropdowns).
